JavaScript

Introduction

JavaScript is an object-oriented programming language used to create interactive effects within web browsers in conjunction with HTML and CSS. The abbreviation that is most commonly used for JavaScript is JS and generally speaking it is necessary for a website to work its best.

Variables

Variables are used in practically every aspect of JavaScript. A variable can be almost anything from a number, boolean, strings, array functions and so on. Variables are one of the first things that are processed before other parts of the code are executed. Generally speaking you want to declare a Variable and give it some sort of value.

Example

var varname1 [= value1] [, varname2 [= value2] ... [, varnameN [= valueN]]];

Example

var x = 1;

if (x === 1) {
var x = 2;

console.log(x);
// expected output: 2
}

console.log(x);
// expected output: 2

Parameters

Function Parameters are the names that are define in the function definition and real values passed to the function in function definition are known as arguments.

Functions

JavaScript functions are the main blocks of code designed to perform a particular task. Functions allow the code to be called many times without repetition.

Example

The general syntax of a function

function name(parameter1, parameter2, parameter3) {
    // code to be executed
}

Example

Simple Function

function show() {
    alert('Hello World!');
}
// expected output: Hello World!

Arrays

In JavaScript, you can use arrays for storing multiple values in a single variable. You can describe an array as a unique variable capable of holding more than one value simultaneously.

Syntax

var array_name = [item1, item2, ...];

Example

var myArray = [1, 2, 3, 4];
var catNamesArray = ["Jacqueline", "Sophia", "Autumn"];
//Arrays in JavaScript can hold different types of data, as shown above.

Associative Arrays

Associative arrays are dynamic objects that the user redefines as needed. When you assign values ​​to keys in a variable of type Array, the array is transformed into an object, and it loses the attributes and methods of Array. Unlike simple arrays, they use curly braces instead of square brackets. This implicitly creates a variable of type Object.

Example

var arr = { "one": 1, "two": 2, "three": 3 };

Associative Array as an Object

var o = new Object();
o["one"] = 1;
o["two"] = 2;
o["three"] = 3;
for(var i in o)
{
     document.write(i + "=" + o[i] + '
');
}


Output:
one=1
two=2
three=3
