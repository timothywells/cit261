HTML5 Tags

Introduction

When learning more about HTML5 I came to the realization that HTML and HTML5 are essentially the same, except that with HTML5 there are more tags that are available. The reason that I came to the conclusion that they are the same is because a vast majority of the sites out there that discuss the difference explain that there once was a difference but because of modern computing HTML5 is the standard but the term HTML is still used to describe both. Additionally, if you are making a new HTML page, HTML5 is automatically loaded/linked to the document. Which in turn makes them one in the same. However, being that there are specific tags that are used, some of which are covered on this page, there are some differences but are hardly noticeable. The tags that are covered on this page will be Audio, Video and Canvas.
